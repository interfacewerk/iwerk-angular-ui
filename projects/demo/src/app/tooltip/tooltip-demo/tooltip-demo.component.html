Tooltip

The tooltip component can be used to display an extra piece of information to an element of an user interface. This piece appears when the mouse is over this element. Add a template that adds the iwTooltip directive inside the element where you want the tooltip to appear.

Positionning

The tooltip is smartly positionned. It will be on top of the target if the target is on the bottom half of the screen. It will be at the bottom of the target if the target is on the top half of the screen.
